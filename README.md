## 📖 Project Overview

**Data Nerd** bridges the gap between job market noise and actionable data. 

Navigating the data science job market is chaotic. Hype often overshadows reality. This platform leverages a live ETL pipeline to scrape, parse, and visualize market demand for data professionals. It helps users prioritize skills based on **volume** and **salary data** rather than Twitter trends.

## ✨ Key Features

* **📊 Real-Time Analytics:** Live dashboard processing data from over 670,000 job postings.
* **💰 Salary Insights:** Correlate specific technical skills with salary benchmarks.
* **📈 Trend Analysis:** Track which tools (Python, SQL, AWS, etc.) are rising or falling in demand.
* **⚡ High-Performance API:** Built on FastAPI for sub-millisecond response times.
